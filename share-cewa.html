🎯 What is SMART BUILDZ?

SMART BUILDZ is a powerful Construction Project Management tool that integrates with Autodesk Construction Cloud to help you manage project phases, generate reports, and collaborate with your team.

🌐 Access URL: http://192.168.1.10:3000
🚀 Open SMART BUILDZ Application

📋 How to Use SMART BUILDZ:

Step 1: Click the "Open SMART BUILDZ Application" button above
Step 2: Enter your Autodesk credentials (Account ID, Client ID, Client Secret)
Step 3: Get your 3-Leg Token using the "Get Token" button
Step 4: Sign in and start managing your projects!

✨ Key Features:

  • 🏢 Hub Management: Select and work with specific Autodesk hubs
  • 📋 Project Selection: Choose individual projects for detailed management
  • 📊 Phase Management: Set and track project phases locally
  • 📈 Report Generation: Create PDF reports for selected projects
  • 🔍 Advanced Filtering: Search and filter projects by phase
  • 👥 Team Collaboration: Share access with team members

🔒 Security & Permissions:

Each user needs their own Autodesk account credentials. SMART BUILDZ respects your organization's permissions - you'll only see projects and hubs you have access to.

💡 Tip: Bookmark this page for easy access to SMART BUILDZ!

🔄 Note: Make sure the SMART BUILDZ server is running on the network.
